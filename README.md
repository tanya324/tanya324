## 🚀 Featured Project

<div align="center">

### 🔷 Skill Graph — Internal Talent Mobility Platform

> *An AI-powered system that matches employees to internal roles using their skills, not just their job titles.*

</div>

| Component | Technology | Purpose |
|---|---|---|
| 🧬 Skill Extraction | BERT + NER | Parse unstructured resumes & profiles |
| 🔍 Semantic Search | SBERT + Pinecone | Vector similarity for role matching |
| 🏗️ Architecture | Microservices | Scalable, modular deployment |
| 📊 Dashboard | React + REST APIs | Visual talent analytics |

```
Resume / Profile
      │
      ▼
  [NER + BERT]  ──→  Skill Entities Extracted
      │
      ▼
  [SBERT Embeddings]  ──→  Pinecone Vector DB
      │
      ▼
  [Cosine Similarity]  ──→  Top Role Matches Ranked
      │
      ▼
  [Analytics Dashboard]  ──→  HR Decision Support
```
